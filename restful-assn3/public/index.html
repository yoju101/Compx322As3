API Documentation

GET /events

Get a list of all events in the database.

Example: http://localhost:3000/events/

POST /events

Create a new event and save it to the database.

Example: http://localhost:3000/events/

Change the drop down to POST and select "raw" and "JSON". Enter in to the test box the info of the new event you want to add.

GET /events/{id}

Get events by ID.

Example: http://localhost:3000/events/2

GET /events/name/{name}

Get events by Name.

Example: http://localhost:3000/events/name/Open Day - Hamilton

PUT /events/{id}

Update event information by ID.

Example: http://localhost:3000/events/1

Change the drop down to PUT and select "raw" and "JSON". Enter in to the test box the info that you want to change.

DELETE /events/delete/{id}

Delete events by ID.

Change GET to DELETE in the drop down

Example: http://localhost:3000/events/delete/3

DELETE /events/delete

Delete all events from the database.

Example: http://localhost:3000/events/delete
